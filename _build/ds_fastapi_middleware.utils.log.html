class ds_fastapi_middleware.utils.log.stdout.Logger(log_level=10)¶

Bases: object

Setup logger for ds. One of the most import aspects of a general logging system is the ability to enforce a common pattern for log messages, in addition to tagging of information, especially when applied to an event driven platform. What we hope to acheive is a uniform use of prefixes in log messages, which will help users to quickly find all log items of an event, across all services and log silos.

Usage: logger = Logger()

# set prefix # 1 handlers are set up here # 1. Stream handler for console # all the messages from daas-service will be in the specified format # [%(asctime)s][%(levelname)s]%(prefix)s[%(module)s]: %(message)s

logger.setup_logger(prefix=”prefix”)

# logging logger.info(msg)

# send log file to s3 logger.send_to_s3(key)

# shutdown logger.shutdown()
