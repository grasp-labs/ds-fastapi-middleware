class ds_fastapi_middleware.AuditMiddleware(app: FastAPI, *, logger, networks: List = [])¶

Bases: BaseHTTPMiddleware

Middleware for creating audit records on all requests made to fastAPI web frameworks.

Middleware secure that on all requests made to api’s, internal and internet facing, a record will be created on which user, tenant and service, how the service was used, from which IP at what time.
